### Conventions
- One **master** document per project lives in `/PROJECT/docs/*_master.md`.
- Use **raw** links for machine access (automations, ChatGPT), and **view** links for humans.
- Commit messages: `feat: …`, `fix: …`, `docs: …`, `chore: …`.
